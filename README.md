# 🚀 TaskFlow Pro

TaskFlow Pro is a simple, modern, and user-friendly task management web application built using **HTML, CSS, and JavaScript**. It helps users efficiently organize, track, and manage daily tasks.

---

## ✨ Features

* ✅ Add new tasks
* ❌ Delete tasks
* ✔ Mark tasks as completed
* 💾 Persistent storage using LocalStorage
* 🔍 Filter tasks (All / Completed / Pending)
* 🌙 Dark mode toggle
* 🎨 Clean and minimal UI

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Browser LocalStorage
